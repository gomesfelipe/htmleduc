Como Jogar

1. Arraste as tags da lista à esquerda para a área de código à direita.

2. Monte um documento HTML válido utilizando as tags de forma correta.

3. Clique no botão "Validar HTML" para verificar se o código está correto.

4. As tags essenciais como <html>, <head>, e <body> devem estar sempre presentes.

5. Para remover as tags, clique no botão "Limpar Tags".

Comandos: CTRL+Z desfaz última ação, CTRL+Y refaz última ação, Ctrl + Shift + X limpa a área de código
